form表单

作用：规定当提交表单时向何处发送表单数据

提交方法有ges与post

input 标签

name 表示该文本输入框名称

maxlength 输入框中允许输入字符的最大数

value 输入框中的默认值

readonly 表示古该框中只能显示，不能修改

label

enctype 规定在表单数据发送之前如何对其编码

三种编码方式：1 application/x-www-form-urlencoded 2 multipart/form-data 3 text/plain

input类型（type）

type=password 密码框输入

type=file 文件上传(使用此类型时编码方式必须为application/x-www-form-urlencoded)

type=checkbox 复选框

type=radio 单选框

type=hidden 隐藏域

type=button 按钮

type=submmit 提交到服务器

type=reset 重置

a标签

作用：控制页面与页面之间的跳转

例如<a href=“http://www.baidu.com”target=_self\>百度

在页面中点击百度标签即可跳转到相应网页

target类型

_self 默认。在当前页面打开链接

_blank 新建一个窗口并在该窗口中打开链接

_parent 在父框架集中打开链接

_top 在整个窗口中打开来链接

锚文本

作用：回到标记的位置

例如 在想回到的位置中添加name=“XXX”

<a href=“#top”>返回

在页面点击返回标签即可回到标记的位置

<captiopn>我的标题<captiopn> 表格带标题

border 边框

width 宽度

height 高度

rowspan
