php函数

函数在程序中可以重复使用

页面加载时函数不会立即执行

函数只有在被调用时才会执行

自定义函数时要以function开头

函数名只能以字母或下划线开头

函数名对大小写不敏感

无参数的函数

function getname(){

return ‘asdf’;

}

echo getname();

有参数的函数

function getname（$name）{

return $name;

}

echo getname(name:‘asdf’);

形参有默认值的有参数的函数

function getname（$name=‘asdf’）{

return $name;

}

echo getname(name:’demo‘);\（输出为demo）

echo getname();\（输出为asdf）

没有参数的匿名函数

$callfunction=function(){

return ‘asdf’；

}；

echo $callfunction();

有参数的匿名函数

$callfunction=function($name){

return $name；

}；

echo $callfunction(name:’asdf’);

isset()函数

作用：判断一个变量是否已定义且不为“NULL“

即同时满足变量存在并且值不为null则返回ture，否则返回false

intvla函数

intval(mixed $var, int $base = 10): in

作用：$var为要转换的值，$base为转换的进制（默认为十进制）

若$var为字符串，他会从字符串的开始部分解析数字，直到遇到非数字字符为止（例如：”123abc“转化为123）

若为浮点数，则会截断小数部分，返回整数部分（例如3.14转化为3）

如果字符串以非数字字符开头，则返回0

空字符串会被转化为0

ture转化为1，flase转化为0

如果传入一个数组，也会返回0（数组不是数字）

null会转化为0

strlen函数

作用：返回字符串长度（包括空格与标点符号）

strlen（）函数的返回值为字符串的字节数

对于单字节编码（如ASCLL），每个字字符通常对应一个字节

对于多字节编码（例如UTF-8），一些字符可能会占用多个字节

若传入空字符串则返回0

若传入一个非字符串，则会将其转化为字符串

substr函数

作用：从一个给定的字符串中提取字符串，需指定提取的位置和长度

substr(string $string, int $start, int $length = ?): string

$string为要截取的原始字符串

$start为开始截取的位置，可以是整数（从字符串开头计数），可以是负数（从字符串末尾计数）

$length为要截取的长度，默认会截取从$start位置到字符串结尾的部分，如果是负数，则表示截取的长度是从末尾向前算

is_array()函数

判断是否为数组，是则返回 1

is_numeric()函数

判断是否为数字，是则返回1

count()函数

判断数组中含有几个元素

运算条件符

array_search函数

在PHP中查找特定的值，会返回找到的值的下标，若没找到则返回false

mixed array_search(mixed $needle, array $haystack, bool $strict = false);

$needle为要查找的元素，可以为任意类型

$haystack为在其中查找的数组

$strict默认为false即进行若比较（==），若为ture则进行强比较（===）

foreach函数

遍历数组

foreach ($array as $key => $value) {
// 对每个元素的操作
}

$array为需要遍历的数组

$key为当前元素的键

$value为当前元素的值
