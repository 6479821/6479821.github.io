修改表的操作

要先选择数据库

修改表名

alter table 旧表名 rename 新表名;

修改字段的数据类型

alter table 表名 modify 属性名 数据类型;

修改字段名

alter table 表名 change 旧属性名 新属性名 新数据类型；

增加字段名

alter table 表名 add 属性名1 数据类型 [完整性约束条件] [（first) | (after属性名2)];

first 将添加的字段放在表头、

after属性名2 将添加的字段放在属性名2之后

删除字段

alter table 表名 drop 删除名

更改表的存储引擎

alter table 表名 engine=存储引擎名

为表中字段添加数据

mysql>insert into 表名(字段名，字段名，)value(要添加的数据)

insert into users(id,username,password)values(1,’moon’,’123456’);

删除表中数据

delete from 表名 [where 条件表达式]；

delete from users where id =1;

若delete语句中没有使用where语句，则会将表中所有数记录删除

更新(修改表中字段的数据)

update 表名 set 字段名1=值1，[，字段名2=值2][where条件表达式]

update users set password=’aaaa’ where id=1;

update users set password=’123456’,username=’moon’ where id=1;

update users set password=456789;

查

select * from 表名 （查询表中所有数据）

select 字段名1，字段名2 from 表名 [where条件表达式] （查询指定的字段）

select username,password from users;

select username,password from users where id=1;

where条件表达式：id=1|id>2|id<9|username=admin|password=123456

in查询

select * from users where id in (3,5,7);

带between and 关键字的查询

select * from users where id between 1 and 10;（输出id在1到10之间所有字段的数据）

select * from users where id not between 1 and 10;（输出id在1到10之外所有字段的数据）

若只需要查询个别字段的数据把*更改成要查询的字段名即可

带distinct关键字的查询

select distinct username from users;（查询users数据库中的字段username中所有不重复的数据）

like查询

select * from users where username like “%m%”;（将users数据库中所有username字段里带m的所有字段的数据输出）

select * from users where username like “m%”;（将users数据库中所有username字段里以m开头的所有字段的数据输出）

select * from users where username like “%m”;（将users数据库中所有username字段里以m结尾的所有字段的数据输出）

下划线_匹配一个字符

select * from users where username like “moo_”;（将users数据库中username字段里所有共四个字符且前三个为moo的所有字段的数据输出）

and查询（需满足所有条件，缺一不可）

select * from users where password=”123456” and username=’admin’;

or查询 （只需要满足任意一个条件）

select * from users where password=”123456” or username=’admin’;

当or与and一起使用时，and的优先级高于or，会优先运算and两边的表达式，再运算or两边的表达式

聚合查询

count()返回某列的行数

select count(*) from users;

sum() 返回某列值的和

avg() 返回某列的平均值

max() 返回某列的最大值

min() 返回某列的最小值

使用limit限制查询结果的数量

select * from users limit m,n;（从第m行开始，不包括第m行，往下输出n行）

select * from users limit 3,2;（输出第4行和第5行）

子查询

where型的子查询（把内层查询结果当作外层查询的比较条件）

select * from users where id in (select id from users where username=”admin”);

from型子查询（将内部查询的结果作为一个表供外部查询）

select * from (select id,username,password from users) where id>2;

select * from (select * from users where id>10) as age_10;（内部查询出来的是一个集合，别名为age_10）

exists型子查询 （将内部查询的结果与外部查询的结果比较，输出公有的）

select * from users exists (select * from users where id>1);

联合查询（两个表的查询）

若需要使用*，两个表的字段需一样多

select * from users union select * from news;

php3

类

在类中，变量称为属性（分为public和private两种，前者可在类的内部，子类，类的外部任意访问和修改，后者只能在定义它的类的内部访问和修改，子类和外部都无法直接修改），函数称为方法，$this关键引用当前对象，并且只在方法内部可用

语法格式：

<?php

class phpclass{

var $var1;

var $var2=”constant string”;

function myfunc($arg1,$arg2){

[…..]

}

[….]

}?>

类使用class关键字加上类名定义

类名后的一对大括号内可定义变量和方法

类的变量使用var来声明，变量也可以初始化值

函数定义类似php函数的定义，但函数只能通过该类及其实例化的对象访问

使用new关键字来调用类

$p1=new phpclass()

使用->来访问类中的成员函数或变量

$p1->myfunc()

魔术方法

php中把以两个下划线开头的方法称为魔术方法

一、__construct()，类的构造函数

php中构造方法是对象创建完成后第一个被对象调用的方法。每个类中都有一个构造方法，如果没有显示的声明它，那么类中都会默认存在一个没有参数且内容为空的构造方法

__construct()：在对象被创建时自动执行的构造方法

构造方法通常用于执行一些有用的初始化任务，如对成员属性在创建对象时赋予初值

一个类中只能声明一个构造方法

构造方法在类中的声明格式：

function __construct([参数列表])

二、__destruct()，类的析构函数

析构方法允许在销毁一个类之前执行一些操作或完成一些功能，如关闭文件

析构函数不能带有任何参数

三，__call()，在对象调用一个不可访问方法时调用

该方法有两个参数，第一个是$function_name 会自动接受不存在的方法名，第二个是$arguments以数组方式接受不存在方法的多个参数

该方法在调用的方法不存在时会自动调用，程序仍会继续执行

四，__callStatic()，用静态方式中调用一个不可访问方法时调用

与__call()大致相同

五，__get，获得一个类的成员变量时调用

六，__set，设置一个类的成员变量时调用

七，__isset(),当对不可访问属性调用isset()或empty()时调用

八，__unset(),当对不可访问属性调用unset()时调用

类的继承

继承就是子类继承父类的特征和行为，使得子类对象（实例）具有父类的属性和方法

PHP中使用关键字extends来继承一个类（class Child extend Parent ）

父类也称为基类

一个父类可以拥有多个子类，一个子类只能拥有一个父类

方法重写

如果从父类继承的方法不能满足子类的需求，可以对其进行改写

访问控制

PHP对属性或方法的访问控制，是通过在前面添加关键字public（公有）、protected（受保护）、private（私有）来实现的

如果用var定义则被视为为public

public：公有的类成员可以在任何地方被访问

protected：受保护的类成员则只能被其自身以及子类和父类访问访问

private：私有的类成员只能被其自身访问

在类中函数可以访问公有、受保护、私有的属性

类的私有和受保护的属性虽然不能在类外直接访问，但类中的公有方法可以访问私有和受保护的属性，因此可以通过在类外调用这个公共方法来间接获取到私有属性的内容

在子类中可以通过关键字parent（parent：：+方法）来访问父类的方法，也可以使用$this关键字

接口

使用接口（interface），可以指定某个类必须实现哪些方法p，但不需要定义这些方法的具体内容

接口是通过interface关键字来定义的，就像定义一个标准的类一样，但其中定义所有的方法都是空的

例如

interface Person{

const NAME=“这个是常量”

public function get_name();

public function set_name($name);

}

接口中所有方法都是公有的，方法可以允许有参数

接口中可以使用常数，但是不能使用变量

要实现一个接口，需使用implements操作符，类中必须实现接口中定义的所有方法，否则会报错，类可以实现多个接口，用逗号来分隔多个接口的名称

例如

class p1 implements Person{

public function set_name($name)

{……}

public function get_name()

{……}

}

抽象类

任何一个类，如果类中至少有一个方法被声明为抽象的，那么这个类就必须被声明为抽象的

被声明为抽象的方法不能有具体的内容，必须为空，可以有参数

定义为抽象的类不能被实例化

使用abstract关键字定义抽象类和方法

继承一个抽象类的时候，子类必须定义父类所有的抽象方法。另外这些方法的访问控制必须和父类中的一样（或者更加宽松）例如某个抽象方法在父类被声明为受保护的，那么在子类中就应该声明为受保护的或者公有的，而不能被声明为私有的

Static关键字

声明类属性或方法为static时，就可以不实例化而直接访问

echo Person(类名)::$name(属性名)；

echo Person(类名)::getname(方法名)；

isset()函数

作用：判断一个变量是否已定义且不为“NULL“

即同时满足变量存在并且值不为null则返回ture，否则返回false

intvla函数

intval(mixed $var, int $base = 10): in

作用：$var为要转换的值，$base为转换的进制（默认为十进制）

若$var为字符串，他会从字符串的开始部分解析数字，直到遇到非数字字符为止（例如：”123abc“转化为123）

若为浮点数，则会截断小数部分，返回整数部分（例如3.14转化为3）

如果字符串以非数字字符开头，则返回0

空字符串会被转化为0

ture转化为1，flase转化为0

如果传入一个数组，也会返回0（数组不是数字）

null会转化为0

strlen函数

作用：返回字符串长度（包括空格与标点符号）

strlen（）函数的返回值为字符串的字节数

对于单字节编码（如ASCLL），每个字字符通常对应一个字节

对于多字节编码（例如UTF-8），一些字符可能会占用多个字节

若传入空字符串则返回0

若传入一个非字符串，则会将其转化为字符串

substr函数

作用：从一个给定的字符串中提取字符串，需指定提取的位置和长度

substr(string $string, int $start, int $length = ?): string

$string为要截取的原始字符串

$start为开始截取的位置，可以是整数（从字符串开头计数），可以是负数（从字符串末尾计数）

$length为要截取的长度，默认会截取从$start位置到字符串结尾的部分，如果是负数，则表示截取的长度是从末尾向前算

作用：得到变量的证书取值

用法：intval(mixed $value,int $base=10):int

其中$value为要转换的整形变量（必填），ture固定转化为1，false固定转化为0

$base为10表示将变量转化为10进制

若$base=0则需观察变量，若字符串包含0x前缀则转化为16进制，若包含0b前缀则转化为2进制，若包含0前缀则转化为8进制，若都不包含默认为10进制

方法1：0e绕过

原理：PHP在处理字符串时会出现缺陷，如果字符串为’5e2’，本来只是一个正常字符串，但PHP会认为这是科学计数法里的e，那么PHP进行比较时会将这个字符串按照科学计数法计算，即5e2=5*10^2=500，由此0e100被认为和0相等。md5加密后的哈希值是一串16进制数，因此需要哈希值第一位为0，第二位为e即可，后面不论是什么都认为和0相等

方法二：数组绕过

原理：无论是PHP弱比较还是强比较，md5()函数无法处理数组，如果传入的是数组，会返回NULL，两个数组经过加密后返回值均为NULL，形成相等

在url后添加?a[]=1&b[]=2;即可

如何绕过

判断校验方式（属于前端校验还是后端校验）

看浏览器是否有给服务器发请求

上传文件后若发现左上角刷新按钮（圈圈）在转，说明向后端发送了请求，即为后端校验，反之为前端校验

前端校验

所有的前端校验等于没有

绕过方式

1 可以右键网页点击检查（F12），点击设置，找到调试程序，勾选停用JavaScript，即可上传其他后缀名的文件

原理：前端校验使用的编程语言为JavaScript，停用JavaScript后校验失效

2 F12找到校验函数删除即可，例：onsubmit=“return checkFile（）”，删除“return checkFile（）”即可（在谷歌与edge浏览器中一般不适用，火狐可用）

后端校验

MIME类型检测

MIME是描述消息内容类型的因特网标准，用来表示文档，文件或字节流的性质和格式。在HTTP数据包中在Content-Type字段显示

常见的MIME类型（通用型）：

1 超文本标记语言.html文件：text/html

2 xml文档.xml：text/xml

3 普通文本.txt：text/plain

4 PDF文档.pdf：application/pdf

5 PNG 图像.png：image/png

6 GIF图形.gif：image/gif

7 JPEG图形.jpeg.jpg：image/jpeg

绕过方式：

1（后端过滤，在服务端对数据包的MIME进行检查） 用bp对网页进行抓包，选择要上传的木马文件，打开bp的拦截功能后点击上传，找到bp拦截到的数据包，找到第二个Content-Type字段，将其内容改为image/png，例如 Content-Type：application/octet-stream改为Content-Type：image/png

2 （后端文件名过滤，禁止上传.asp|.aspx|.php|.jsp后缀文件） 用bp对网页进行抓包，选择要上传的木马文件，打开bp的拦截功能后点击上传，找到bp拦截到的数据包，找到Content-Dispositoin，在文件明后添加：：$DATA

若源代码中存在函数检验：：$DATA若发现则替换其为空字符，则可以进行双写

访问url时要将url中的$DATA删除才可正常访问

post11.15(sql注入)

#一 SQL介绍

1 SQL全称是结构化查询语言，是一种特殊目的的编程语言，是一种数据库查询和程序设计语言，用于存取数据以及 查询、更新和管理关系数据库系统，同时也是数据库脚本文件的扩展名。

2 SQL注入攻击指的是通过构建特殊的输入作为参数传入Web应用程序，而这些输入大都是SQL语法里的一些组合，通过执行SQL语句进而执行攻击者所要的操作，其主要原因是程序没有细致地过滤用户输入的数据，致使非法数据侵入系统。

#二 工具:Sqlmap

1 Sqlmap全面支持六种SQL注入技术：

（1） 基于布尔类型的盲注：即可以根据返回页面判断条件真假的注入。

（2） 基于时间的盲注：即不能根据页面返回的内容判断任何信息，要用条件语句查看时间延迟语句是否已执行(即页面返回时间是否增加)来判断。

（3） 基于报错注入：即页面会返回错误信息，或者把注入的语句的结果直接返回到页面中。

（4） 联合查询注入：在可以使用Union的情况下的注入。

（5） 堆查询注入：可以同时执行多条语句时的注入。

（6） 带外注入：构造SQL语句，这些语句在呈现给数据库时会触发数据库系统创建与攻击者控制的外部服务器的连接。以这种方式，攻击者可以收集数据或可能控制数据库的行为。

2 使用 Sqlmap 对一个注入点进行测试时，可以使用 -v x 参数指定回显信息的复杂度，x范围为[0~6]，共有 7 个等级，默认为 1。

  • 0：只显示python错误以及严重的信息。
  • 1：同时显示基本信息和警告信息。（默认）
  • 2：同时显示debug信息。
  • 3：同时显示注入的payload。
  • 4：同时显示HTTP请求。
  • 5：同时显示HTTP响应头。
  • 6：同时显示HTTP响应页面。
  • 如果想看到 Sqlmap 发送的测试 payload 最好的等级就是3，例：sqlmap.py -v 3 -u “http://www.xxx.com/a.php?id=x“

#三 sqlmap的使用

1 判断是否存在sql注入：

假设目标注入点是 http://127.0.0.1/sqli-labs/Less-1/?id=1，判断其是否存在注入的命令如下：

sqlmap.py -u http://127.0.0.1/sqli-labs/Less-1/?id=1
